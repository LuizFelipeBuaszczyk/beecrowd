# Beecrowd
Este repositório contém resoluções dos exercícios de **leetcode** disponíveis na plataforma Beecrowd. As soluções dos exercícios são feitas por mim (Luiz Felipe Buaszczyk). O repositório tem o intuito **acadêmico** onde estudantes podem verificar respostas e entender a resolução de exercícios. 

Todas as resoluções apresentadas neste repositório foram formuladas em vídeo no canal do youtube [Luiz Felipe Buaszczyk](https://www.youtube.com/@luizfelipe_tech).
