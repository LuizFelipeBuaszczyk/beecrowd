# TDA Racional

O objetivo deste exercício é realizar operações com os números fracionais.

## Solução

Para solucionar este exercício é necessário dar entrada em um valor `n` que será utilizado para indicar a quantidade de entradas que serão realizadas. Após isso, é realizado a entradas das operações seguindo o **sempre** padrão `n1Nominador >> '/' >> n1Denominador >> operador >> n2Nominador >> '/' >> n2Denominador`. É necessário validar a operação a ser realizada, e com base nisso executar a respectiva fórmula passada. Após realizar o cálculo é necessário achar o valor simplificado e a saída deve ter o primeiro resultado e o valor simplificado.
