# Area
Este exercício tem o objetivo de realizar algumas formas matemáticas e imprimir o seu resultado com 3 casas decimais.

## Solução
Para resolver este exercício é necessário dar entrada em três valores double, `A`, `B` e `C`. Também é interessante declarar uma constante `PI = 3.14159`. Após é necessário executar as formulas e realizar o output, com o nome da geometria e o seu resultado com 3 casas decimais.

### Formulas
- Area do Triângulo: `area = (A * C) / 2`
- Area do Circulo: `area = PI * pow(C, 2)`
- Area do Trapezio: `area = ((A + B) * C) / 2`
- Area do Quadrado: `area = pow(B, 2)`
- Area do Retângulo: `area = A * B`
