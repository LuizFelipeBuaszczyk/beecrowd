# Distância
O exercício consiste em calcular o tempo necessário para o carro Y alcançar o carro X, levando em consideração que o carro Y fica 1 km mais perto a cada dois minutos.

## Solução
Para resolver o problema é possível utilizar um loop while para simular o processo de aproximação entre os dois carros. A cada iteração do loop, a distancia é diminuida em 1 km e o tempo é incrementado em 2 minutos. Quando o valor for menor que 0 ou igual a zero o loop é interrompido e o tempo total é exibido.
