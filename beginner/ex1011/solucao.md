# Sphere
Este exercício tem o objetivo de calcular o volume de uma esfera dada a entrada do raio, levando em consideração `PI = 3.14159`

## Solução
Para resolver é necessário dar entrada em um double que será o raio, após isso é necessário realizar a fórmula `volume = (4/3) * PI * pow(raio, 3)`, no caso da linguagem C++ é necessário atribuir dos valores da primeira divisão com casa decimal (4.0/3 ou 4/3.0), para quando o programa rodar ele tratar a divisão com ponto flutuante e nao inteira.
