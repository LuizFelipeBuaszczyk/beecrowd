# Hello World!

Este é o exercício mais simples da plataforma.

Seu intuito é proporcionar ao programador um primeiro contato com a sintaxe da linguagem escolhida e, além disso, quebrar a famosa "maldição do Hello World!".

## Solução
Para solucionar o problema é necessário realizar a operação de **output** com a mensagem "Hello World!".
